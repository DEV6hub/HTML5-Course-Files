<p>

hover here to switch two property values
instantly

<p class="transitionable">

hover here to switch two property values
gradually


		
	

	
			p	{
				/* starting state */
				margin-left: 100px; opacity: .25;
			}
			p:hover {
				/* ending state */
				margin-left: 300px; opacity: 1;
			}
			.transitionable {
				/* HOW the margin-left property changes */
				-webkit-transition: margin-left 0.5s ease-in 0.5s, opacity 0.5s ease-in 1s;
			}
